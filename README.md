# Telegram Finance Bot

Bot pessoal de controle financeiro desenvolvido em Python usando Telegram API.

## Funções
- registrar gastos
- registrar entradas
- relatório mensal
- cálculo de ciclo de cartão
- alertas de saldo

## Tecnologias
- Python
- SQLite
- python-telegram-bot
- APScheduler
